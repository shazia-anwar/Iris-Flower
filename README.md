# Iris Flower Classification

## Project Overview
This project classifies Iris flowers into three species — Iris-setosa, Iris-versicolor, and Iris-virginica — using sepal and petal measurements.

Multiple machine learning algorithms were implemented and compared to evaluate performance on a multi-class classification problem.

---

## Dataset Description
The dataset contains 150 samples with the following features:

- Id – Unique identifier
- SepalLengthCm – Sepal length in centimeters
- SepalWidthCm – Sepal width in centimeters
- PetalLengthCm – Petal length in centimeters
- PetalWidthCm – Petal width in centimeters
- Species – Target variable (flower species)

There are three classes:
- Iris-setosa
- Iris-versicolor
- Iris-virginica
